# NZXT Website Clone

![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white)

A responsive, pixel-perfect front-end clone of the official NZXT website. This project was built entirely from scratch to practice modern layout techniques, focusing on the brand's signature sleek, dark-themed aesthetic and hardware product showcases.

## 🎯 Features

* **Premium Dark UI:** Accurately replicates the modern, high-contrast dark theme typical of the PC gaming industry.
* **Complex Grid Layouts:** Utilizes CSS Grid to recreate intricate product galleries and promotional banners.
* **Responsive Design:** Fully adaptable layout that provides a seamless browsing experience across desktop, tablet, and mobile devices.
* **Smooth Micro-interactions:** Features custom CSS transitions and hover states for buttons, navigation links, and product cards.
* **Zero Dependencies:** Built strictly with vanilla HTML5 and CSS3—no external frameworks like Tailwind or Bootstrap.

## 🚀 Technologies Used

* **HTML5** (Semantic structuring)
* **CSS3** (Flexbox, Grid, Custom Properties/Variables, Animations)

## 📁 Getting Started

To view this project locally on your machine, follow these simple steps:

### Prerequisites
You only need a modern web browser (Chrome, Firefox, Safari, Edge) to view this project. No Node.js or build tools are required.

### Installation

1. Download the project files to your local machine and extract the folder.
2. Navigate into the extracted project directory.
3. Simply double-click the `index.html` file to open it in your default web browser. 
   * *(Alternatively, if you are using VS Code, you can use the **Live Server** extension for hot-reloading as you explore the code).*
